    Ocellus

    June 13, 2021

    Ocellus is a frontend to the Biodiversity Literature Repository (BLR) community with images on Zenodo, taxonomic treatments in TreatmentBank, and citations on RefBank. Ocellus depends on Zenodeo, a nodejs API that queries, analyzes and aggregates results from these various repositories via a single, unified interface.
